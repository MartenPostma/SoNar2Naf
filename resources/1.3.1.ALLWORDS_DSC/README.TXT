DESCRIPTION:

These are three XML file for each Part-of-speech containing the annotated examples for the allwords corpus 
developed in the DutchSemCor project. For each annotated example (element <token>) the following information
is stored in the XML file:
 + lemma: the lemma of the example
 + pos: the part-of-speech
 + sense_id: the lexical unit identifier of Cornetto for the example
 + token_id: the token identifier of the example in SONAR
